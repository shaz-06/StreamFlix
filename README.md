# 🎬 StreamFlix – Netflix Style Streaming Platform

StreamFlix is a Netflix-inspired movie streaming interface built using **HTML, TailwindCSS, and JavaScript** with movie data powered by the **TMDB API** and trailers powered by **YouTube API**.

🌐 **Live Demo**  
https://stream-flix-xi.vercel.app/

---

## 🚀 Features

- 🎥 Dynamic movie browsing using TMDB API  
- 🔎 Search movies, TV shows, and people  
- 🎬 Watch trailers powered by YouTube  
- ⭐ Netflix-style hero section  
- 🎞 Interactive movie cards with hover effects  
- 📺 Movie detail modal with additional information  
- 👤 Profile dropdown with account options  
- 📱 Fully responsive UI

## ⚡ How It Works

1. The application fetches movie data from **TMDB API**.
2. Movies are dynamically rendered into rows.
3. Clicking a movie opens a **detail modal** with additional information.
4. Trailers are fetched and played using the **YouTube IFrame API**.
